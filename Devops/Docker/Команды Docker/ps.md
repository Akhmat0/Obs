- **Описание**: Список запущенных контейнеров.
- **Синтаксис**:- bash
    
    `docker ps [OPTIONS]`
    
- **Основные флаги**:
    
    - `-a` или `--all`: Показывает все контейнеры (включая остановленные).
    - `-q` или `--quiet`: Показывает только идентификаторы контейнеров.
    
- **Пример использования**:bash

`docker ps -a`
